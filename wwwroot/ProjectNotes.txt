What this project does currently:

- Stores my recipes in a database.
	- database built out with future enhancements in mind.

- Displays them as a gallery to choose from.

- Displays ingredients and simplified instructions on single page with minimal to no scrolling.


What this project will do next/eventually:

- Have a single input page.

- Single page editing of a recipe.

- Allow a series of recipes to be chosen for a week, and email a shopping list.

- Convert from various measures to grams for more accurate cooking.

- Calculate nutrition data.
